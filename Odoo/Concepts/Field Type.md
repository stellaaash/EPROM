Field types **define the types of fields inside an Odoo [[Model]].**

# List of Field Types
All field types can have a **default value**.
Suffixes of fields allow you to set a unit for the field's values. For example *Cost per km*.
- **Text**
- **Multiline Text**
- **HTML**
- **Checkbox**
- **Integer**
- **Decimal**
- **Monetary**
- **Date**
- **Date & Time**
- **Selection**: Selection of a value from a group of predefined values
- **Tags**
- **Many2one**: Selection of a single record from another model
- **Many2many**: Selection of multiple records from another model
- **Separator**: Group several properties under a foldable label
