System V refers to a **major version of [[Unix]] released by [[AT&T]] in 1983**.
# Inter-Process Communications
System V is often mentioned when talking about [[InterProcess Communication]] in a [[Unix]] system.
## Message Queues
Allow processes to exchange data in discrete messages.
## Semaphores
Synchronization primitives for coordinating access to shared resources (think [[Mutex]], but more complicated)
## Shared Memory
Simply allows multiple processes to access the same memory region.
