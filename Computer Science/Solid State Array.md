SSA or *Solid State Array* is a **storage array built entirely from [[Solid State Drive]]s and used as shared storage for servers**.
It is a chassis or appliance that contains multiple SSDs managed **as a single storage system**.
It typically uses [[RAID]], controllers, cache and management software.
It is used to **present logical volumes or [[LUN]]s to servers over [[SAN]] or [[NAS]] protocols**. Widely used in datacenters.
