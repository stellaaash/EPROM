An open-source relational database management system.
A relational database organizes data into **one or more data tables** in which data may be related to each other.
These relations help structure the data.
It is often running on top of an [[Operating System]], implementing its storage in the computer's and running hand-in-hand with the OS to allow for network access and many other features.
# SQL as a language
SQL is a language that programmers use to create, modify and extract data from relational databases.
It is also used to control user access to that same database.
