The TCP/IP model is the [[OSI Model]] implemented using the [[Internet Protocol]] over [[Transmission Control Protocol]].
# Layers
## Application Layer
In OSI, this equals to the application, presentation and session layers. 
## Transport Layer
OSI layer 4.
## Internet Layer
This is layer 3, called Network in OSI.
## Link Layer
This is OSI layer 2.
